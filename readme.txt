Simple plugin to save contact form 7 data. Simple but flexible.

== Description ==

This is a simple plugin that saves contact form 7 entries into the database for later use.

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* write any user personal data to the database;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `save-cf7-entry` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
3. You will find **CF7 Entries** menu in your WordPress admin screen where you can manage them.
